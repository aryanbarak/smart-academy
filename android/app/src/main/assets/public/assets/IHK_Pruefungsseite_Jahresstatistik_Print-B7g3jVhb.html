1) Aufgabenstellung (IHK‑Stil)

Entwickeln Sie eine Funktion ermittleJahresstatistik(int[][] verbrauch), die für ein Kalenderjahr den minimalen und maximalen Monatsverbrauch ermittelt. Das zweidimensionale Array verbrauch[monat][tag] enthält die Tageswerte je Monat. Geben Sie das Ergebnis als Objekt der Klasse Jahresstatistik zurück.

Funktion ermittleJahresstatistik(verbrauch : Integer[][]) : Jahresstatistik
    // Schutz: leeres Jahr → 0/0
    Wenn verbrauch.length = 0 dann
        Rückgabe NEUE Jahresstatistik(0, 0)
    Ende Wenn

    Deklariere min : Integer = MAX_WERT
    Deklariere max : Integer = MIN_WERT

    // Äußere Schleife: Monate
    Für m = 0 bis verbrauch.length - 1
        summe = 0

        // Innere Schleife: Tage summieren
        Für t = 0 bis verbrauch[m].length - 1
            summe = summe + verbrauch[m][t]
        Ende Für

        // Min/Max aktualisieren
        Wenn summe < min dann min = summe Ende Wenn
        Wenn summe > max dann max = summe Ende Wenn
    Ende Für

    // Objekt mit beiden Werten zurückgeben
    Rückgabe NEUE Jahresstatistik(min, max)

5) Kurzbeschreibung (Programmlogik)

Die Funktion iteriert über alle Monate, summiert deren Tageswerte zu einer Monatssumme und vergleicht diese mit den bisher gefundenen Minimal- und Maximalwerten. Abschließend werden beide Werte in einem Objekt Jahresstatistik gebündelt und zurückgegeben.
